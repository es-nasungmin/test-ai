## 월 10,000 채팅 기준

### OpenAI API 비용

#### 1. 텍스트 임베딩 (embedding)
- 서비스: `text-embedding-3-small`
- 사용량: 10,000회 × 평균 600 tokens (질문 200 + 이전 대화 user 2턴 × 200) = **6M tokens**
- 단가: $0.02 / 1M tokens
- **월 비용: $0.12**

#### 2. 챗봇 답변 생성 (gpt-4o-mini)
- 서비스: `gpt-4o-mini`
- Input (질문 + 컨텍스트 + 과거 대화)
  - 10,000회 × 평균 1,500 tokens = **15M tokens**
  - 단가: $0.15 / 1M tokens
  - 비용: **$2.25**
- Output (답변)
  - 10,000회 × 평균 600 tokens = **6M tokens**
  - 단가: $0.60 / 1M tokens
  - 비용: **$3.60**
- **Input + Output 월 비용: $5.85**

#### 3. LLM Reranking (gpt-4o-mini, 최대 기준)
- 매 질문마다 reranking 발생 가정 (고신뢰 skip 조건 미적용 최악 시나리오)
- Input: 10,000회 × 800 tokens (시스템+질문+후보 8개×90) = **8M tokens**
  - 단가: $0.15 / 1M tokens
  - 비용: **$1.20**
- Output: 10,000회 × 180 tokens (max_tokens 설정 기준) = **1.8M tokens**
  - 단가: $0.60 / 1M tokens
  - 비용: **$1.08**
- **Input + Output 월 비용: $2.28**

### 기타 인프라 비용

#### Qdrant (벡터 데이터베이스)
- **자체 호스팅 (온프레미스)**: 무료
  - 기존 Windows Server에 Docker 또는 직접 설치
  - 초기 설정 후 운영 비용 없음
  - 저장소 및 메모리: 기존 인프라 범위 내
- 클라우드 (Qdrant Cloud):
  - 스탠다드 플랜: ~$30/월 (₩36,000)
  - 스케일 요금제: 용량/쿼리에 따라 변동

#### 서버 호스팅 (IIS + MSSQL)
- **기존 온프레미스 서버**: 무료
  - 이미 운영 중인 Windows Server + IIS + MSSQL 활용
  - 추가 서버 비용 없음
  - 단, Qdrant를 같은 서버에 배포 시 리소스 모니터링 필요
- 신규 클라우드 구축:
  - Azure:
    - App Service (B1): ~$10/월 (₩12,000)
    - SQL Database (S0): ~$15/월 (₩18,000)
    - 합계: ~$25/월 (₩30,000)
  - AWS EC2 + RDS: ~$30-50/월 (₩36,000-60,000) (스팩에 따라)

## 비용 최적화 팁

1. **임베딩 캐싱**: 동일한 질문의 임베딩 결과를 재사용 (RAG 단계에서 이미 적용)
2. **컨텍스트 최소화**: 과거 대화 턴(historyTurnCount) 조정으로 토큰 절감
3. **온프레미스 운영**: 대규모 운영 시 자체 서버 구축으로 인프라 비용 절감
4. **배치 처리**: 비즈니스 시간 외 대량 질문은 배치로 처리 (별도 요금 가능)
5. **Qdrant 자체 호스팅**: 충분한 용량이 있으면 온프레미스로 운영
