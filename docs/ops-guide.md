# AI 챗봇 운영 가이드 (운영팀용)

## 이 시스템이 하는 일

고객이 챗봇에 질문을 입력하면, AI가 운영팀이 등록해 둔 자료(KMS)에서 가장 적합한 답변을 찾아 자동으로 응답합니다.  
운영팀은 AI가 잘 답할 수 있도록 자료를 관리하고, 미흡한 답변을 보완하는 역할을 합니다.

---

## 운영팀이 할 일 (딱 두 가지)

### 1. KMS 자료 등록·수정
- 고객이 자주 묻는 질문과 그에 맞는 답변 내용을 KMS에 등록합니다.
- 정책이나 내용이 바뀌면 해당 항목을 수정합니다.

### 2. 미흡 답변 확인 및 보완
- AI가 자신 없어 한 질문 목록을 주기적으로 확인합니다.
- 반복적으로 등장하는 질문이 있으면 KMS에 자료를 추가합니다.

---

## 화면별 사용법

### KMS 관리
답변 자료를 등록·수정·삭제하는 핵심 화면입니다.

| 기능 | 방법 |
|------|------|
| 새 자료 등록 | 우측 상단 **KMS 작성** 버튼 클릭 |
| 자료 수정 | 목록에서 항목 클릭 → 내용 수정 후 저장 |
| 자료 삭제 | 항목 선택 후 삭제 버튼 |
| 대량 등록 | **CSV 가져오기** 버튼으로 엑셀 기반 일괄 등록 |
| 검색 | 제목·내용·키워드로 검색 가능 |

**KMS 작성 시 채워야 할 항목**

| 항목 | 설명 | 필수 |
|------|------|------|
| 제목 | 이 자료가 다루는 주제 (예: 비밀번호 변경 방법) | ✅ |
| 내용 | 실제 답변 내용 | ✅ |
| 예상 질문 | 고객이 이런 식으로 물어볼 것 같은 질문 (여러 개 가능) | 권장 |
| 키워드 | 검색에 도움이 되는 핵심 단어 | 권장 |
| 플랫폼 | 어느 서비스에 해당하는 자료인지 | ✅ |
| 공개 범위 | 고객용(user) / 상담원용(admin) 구분 | ✅ |

> **팁:** 예상 질문을 많이 넣을수록 AI 답변 적중률이 높아집니다.  
> 예) "비밀번호를 잊어버렸어요", "로그인이 안 돼요", "패스워드 재설정 방법"

---

### 채팅관리
실제 고객과의 대화 이력을 확인하는 화면입니다.

- 날짜·플랫폼별 필터로 원하는 대화 조회
- 특정 대화를 열어 AI가 어떤 자료를 참조했는지 확인 가능

---

### 질문 패턴 분석
AI가 낮은 자신감으로 답변한 질문 목록입니다.

- **여기 자주 뜨는 질문 = KMS에 자료가 부족하다는 신호**
- 반복 출현하는 질문을 발견하면 KMS에 관련 자료를 추가하세요.

---

## 자주 묻는 질문

**Q. 자료를 등록하면 언제부터 챗봇에 반영되나요?**  
등록 즉시 자동으로 처리되며 수 초 내 반영됩니다.  
KMS 목록에서 벡터 동기화 상태가 **synced**로 바뀌면 완료된 것입니다.

**Q. AI가 엉뚱한 답변을 하면 어떻게 하나요?**  
채팅관리에서 해당 대화를 찾아 AI가 참조한 자료를 확인합니다.  
자료 내용이 부정확하면 KMS에서 수정하거나, 관련 예상 질문을 추가합니다.

**Q. 여러 자료를 한 번에 등록할 수 있나요?**  
KMS 관리 화면의 **CSV 가져오기** 기능을 사용하면 됩니다.  
CSV 양식은 담당 개발팀에 요청하거나 기존 자료를 내보내서 참고하세요.

**Q. 삭제한 자료는 복구되나요?**  
삭제 후 복구는 불가합니다. 삭제 전 반드시 확인하세요.  
비활성화가 필요한 경우 삭제 대신 내용을 수정하거나 담당자에게 문의하세요.
